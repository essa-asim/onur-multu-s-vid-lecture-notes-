## Programmable Logic Array (PLA)

- For n input logic functions  
- We need a PLA with AND gates  

### What is it?
- Type of digital hardware chip  
- Used to build complex combinational logic circuits  

---

## Two-Level Architecture
- Programmable AND array (product)  
- Programmable OR array (sum)  

-> Used for implementing any collection of logic functions  

---

## Applications
- Implementation of full adders using PLA
